## 진행 내용

- 프론트 더미 데이터 토글 추가
  - `VITE_USE_MOCKS=true|false`
  - 더미 비활성화 시 빈 상태 화면 표시
- 프론트 상호작용 보강
  - 프로젝트 검색/필터/정렬/보기 전환
  - 새 분석 옵션 선택
  - 상세 분석 탭 전환
- CSS 구조 정리
  - `global.css`는 reset과 변수만 유지
  - 페이지/레이아웃/컴포넌트별 CSS 분리
- 백엔드 구조 조정
  - `app/` 패키지 중심 구조에서 루트 폴더형 구조로 전환
  - `main.py`, `routes/`, `schemas/`, `core/`, `utils/` 기준
  - `fastset.py`를 `utils/fastset.py`로 이동
- 백엔드 설정 방식 변경
  - `.env` 값은 `core/settings.py`에서 읽음
  - 코드에서 직접 `os.getenv`를 호출하지 않도록 정리
- 백엔드 로컬 실행 구조 검증
  - `uvicorn main:app --reload` 기준으로 구조 변경
  - `utils/fastset.py`에서 라우터 등록
- 차트 라이브러리 추가
  - `recharts` 추가
  - 홈, 결과 요약, 분석 내역, 커밋 메시지 페이지에 차트 적용
  - Vite `manualChunks`로 차트 번들 분리
- 문서 정상화
  - 깨진 한글 문서 재작성
  - 현재 백엔드/프론트 구조 반영
