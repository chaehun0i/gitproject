### CSS 분리 메모

- 이번 작업에서는 마이페이지 전용 스타일을 우선 분리했다.
- 나머지 내부 페이지 스타일은 아직 `appPages.css`에 공통/페이지 규칙이 섞여 있어, 다음 CSS 정리 작업에서 페이지 단위로 추가 분리할 예정이다.

### 검증

- 마이페이지와 전용 CSS에서 `프로필 사진`, `개발자 토큰`, `DB`, `Redis`, `PostgreSQL`, `MariaDB`, `WebSocket`, `Git Parser`, `Diff Parser` 직접 노출이 검색되지 않았다.
- `npm.cmd run build` 통과.

### 남은 개선점

- `appPages.css`에 남아 있는 Home, 프로젝트, 분석, 히스토리 계열 스타일을 페이지별 CSS로 추가 분리해야 한다.
- 실제 브라우저에서 각 모달의 높이와 모바일 스크롤감을 확인하면 좋다.
