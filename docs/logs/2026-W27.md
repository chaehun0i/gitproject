### 추가 보강

- 분석 진행, 결과 요약, 상세 분석, 커밋 메시지, 분석 내역, 설정 화면에 남아 있던 사용자에게 과한 기술명 노출을 사용자 중심 문구로 바꿨다.
- 사이드바는 전달받은 로고 이미지 안에 `CommitLens` 텍스트가 포함되어 있어 별도 브랜드 텍스트와 설명을 숨김 처리했다.
- 최종 검색 기준으로 프론트 페이지/컴포넌트의 일반 사용자 화면 문자열에서 `Git Parser`, `Diff Parser`, `DB`, `Redis`, `PostgreSQL`, `WebSocket`, `AI Runtime`, `MariaDB`, `Local Runtime` 직접 노출이 잡히지 않음을 확인했다.
- 최종 `npm.cmd run build` 통과.
