# 이미지 자산

프로젝트 로고, 화면용 이미지, 안내 이미지 등을 둡니다.

## 현재 사용 이미지

- `commitlens-logo.png`: 랜딩, 인증 화면, 사이드바 등 브랜드 영역에 사용하는 CommitLens 로고
- `commitlens-hero.png`: 랜딩 Hero 시각 요소
- `commitlens-product-visual.png`: 로그인/Workspace 등 제품 화면 분위기를 보강하는 SaaS형 제품 비주얼

## 관리 기준

- 실사 이미지나 사람 사진은 사용하지 않습니다.
- 서비스 성격에 맞는 코드 분석, 저장소, 커밋 흐름, AI 요약 느낌의 추상 이미지를 사용합니다.
- 이미지를 추가하면 사용 위치와 목적을 주간 로그에 기록합니다.
